# Userfeeds-Widgets

JS widget library

## Development
### Requirements
- [nodejs](https://nodejs.org/en/download/current/) (recomended >= 8.0.0)
- yarn - [recomended way](https://yarnpkg.com/en/docs/install#alternatives-tab)

### Instalaction
- clone this repository
- `cd JS-Widgets`
- `yarn`

### Run
- `yarn start` - browser should open automaticly
